# Toon Website
<br>

## 프로젝트 소개
- React를 사용하여 개발한 만화 홈페이지의 프론트엔드 입니다.
- REST API로 백엔드와 연동할 수 있도록 API통신 부분을 구현했습니다.

## 주요 기능
### 공통 기능
- 로그인, 회원가입
- 별점 
- 즐겨찾기
- 댓글
- 요일별 만화
### 관리자 기능
- 만화 등록, 수정, 삭제
- 에피소드 등록, 수정, 삭제
